Journey to the Center of Hawkthorne

This project is a derivative of Kyle Conroy's Journey to the Center of Hawkthorne project. Journey to the Center of Hawkthorne is a sidescrolling game based on community's Digital Estate Planning episode.

Where the main project focuses on single player campaign, this is multiple projects all designed with the aim of supporting multiplayer over a network.
